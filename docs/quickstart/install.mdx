## Pre-requisites

- [Neovim 0.8.3](https://github.com/neovim/neovim/releases/tag/v0.8.0).
- [Nerd Font](https://www.nerdfonts.com/) Set it in your terminal emulator.
  - Make sure the nerd font you set doesnt end with <strong>Mono</strong> to prevent small icons. 
   - <strong>Example : </strong> Iosevka Nerd Font and not <s>Iosevka Nerd Font Mono</s>
- [Ripgrep](https://github.com/BurntSushi/ripgrep) is required for grep searching with Telescope <strong>(OPTIONAL)</strong>. 
- GCC, Windows users must have [`mingw`](http://mingw-w64.org/downloads) installed and set on path.
- Delete old neovim folder (check commands below)

## Uninstall
```bash
# linux/macos (unix)
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim

# windows
rd -r ~\AppData\Local\nvim
rd -r ~\AppData\Local\nvim-data
```
